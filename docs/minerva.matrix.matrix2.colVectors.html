colVectors

minerva matrix matrix2

Extract the column vectors of a given matrix. The vectors are read like this: a b c d => (a, c), (b, d)

pure nothrow @safe
Vector2[]
colVectors
(
Matrix2x2 m
)

Parameters

m Matrix2x2

is the matrix from which the column vectors should be be extracted.

Return Value

Type: Vector2[]

A Vector2[] containg all the column vectors.

Examples

Get the col vectors of a matrix

auto a = Matrix2x2(1.0, 3.0, -4.0, 2.0);
assert(a.colVectors() == [Vector2(1.0, -4.0), Vector2(3.0, 2.0)]);
