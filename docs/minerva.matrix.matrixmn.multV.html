multV

minerva matrix matrixmn

Multiplies a MxN matrix and a N dimensional vector. You have to specify the dimensions of the matrices in the multiplication template arguments like multV!(M,N)(lhs, rhs).

Vector!M
multV
(
size_t M
size_t N
)
(
Matrix!(M, N) lhs
,
Vector!N rhs
)

Parameters

lhs Matrix!(M, N)

A MxN matrix

rhs Vector!N

A N dimensional vector

Return Value

Type: Vector!M

The resulting M dimensional vector.

Examples

Matrix-Vector multiplication

Matrix!(2, 3) m1 = Matrix!(2, 3)([
    [1.0, -1.0, 2.0],
    [0.0, -3.0, 1.0]
  ]);

Vector!3 v1 = Vector!3([2.0, 1.0, 0.0]);

assert(m1.multV!(2, 3)(v1).components == [1.0, -3.0]);
