shiftLeft

minerva vector vector3

Shifts the x, y and z values of v to the left, so x becomes y, y becomes z and z becomes x.

pure nothrow @nogc @safe
Vector3
shiftLeft
(
Vector3 v
)

Parameters

v Vector3

is the vector we want to swap to the left.

Return Value

Type: Vector3

A Vector3 with swapped components in comparison to v.

Examples

Shifts the components to the left

auto a = Vector3(5.0, 0.0, 3.0);
assert(a.shiftLeft() == Vector3(0.0, 3.0, 5.0));
